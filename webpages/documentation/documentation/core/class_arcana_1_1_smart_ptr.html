Detailed Description

template<typename ObjectType = Object>
class Arcana::SmartPtr< ObjectType >

A templated smart pointer implementation.

Smart pointers wrap raw pointers, deleting the raw pointers upon destruction. This eliminates the need to call 'delete' on raw pointers and can prevent memory leaks.
