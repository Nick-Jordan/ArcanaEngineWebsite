Detailed Description

template<typename ContainerType, typename T, typename IndexType>
class Arcana::IndexedContainerIterator< ContainerType, T, IndexType >

Iterator for custom ArcanaEngine containers.

Array iterators are created easily with Array.createIterator() or Array.createConstIterator(). This class provides operators for incrementing or decrementing the array index. Using the dereference operator returns a reference to the array element at the current index.
