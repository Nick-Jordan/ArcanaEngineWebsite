Detailed Description

template<typename ObjectType, typename ReturnValue, typename... ArgumentTypes>
class Arcana::MemberFunctionCallbackInstance< ObjectType, ReturnValue, ArgumentTypes >

Callback instance wrapping a member function pointer.

Needs to be bound to a function pointer and an object.
