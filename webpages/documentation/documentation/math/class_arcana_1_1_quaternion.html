Detailed Description

template<typename T>
class Arcana::Quaternion< T >

A quaternion object mainly used for representing rotations.

A quaterion is a mathematical object represented by a 4-vector that behaves according to normed division albebra over the real numbers. q = [qw, qx, qy, qz] Rotation quaternions are defines as follows: q = [cos(theta/2), a * sin(theta/2)], where a is the axis of rotation. Quaternions can be converted to matrices for use with OpenGL.
